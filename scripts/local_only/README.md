# Локальные скрипты (не для VPS)

Скрипты в этой папке предназначены только для локального использования. Они не входят в логику приложения на VPS и не вызываются при деплое.

- **run_embedded_pdf_extract.py** — извлечение встроенного текста из PDF (файл с «ocr» в имени) без БД и без Tesseract. Результат пишется в `data/ocr_raw/` и `data/ocr_normalized/`. Готовый файл можно подгрузить на VPS по инструкции в `docs/UPLOAD_NORMALIZED_TO_VPS.md`.
