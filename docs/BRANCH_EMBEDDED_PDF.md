# Ветка feature/embedded-pdf: кликабельный PDF без OCR

Эта ветка добавляет поддержку **кликабельного PDF** (с встроенным текстом): файл с припиской «ocr» в имени в той же папке, что и исходный PDF, обрабатывается **без Tesseract** — текст извлекается через `get_text(sort=True)` (PyMuPDF). Подходит для тяжёлых PDF (например ~200 МБ), переведённых из скана в кликабельный вид.

- **ingestion.py:** `_resolve_pdf_path_with_ocr_variant()` выбирает файл с «ocr» в имени; при таком файле используется извлечение встроенного текста вместо OCR.
- **scripts/run_embedded_pdf_extract.py:** локальное извлечение без БД (для устройства без пуша на GitHub). Результат — `data/ocr_raw/` и `data/ocr_normalized/`; дальше можно подгрузить .md на VPS по инструкции в `docs/UPLOAD_NORMALIZED_TO_VPS.md`.

На **main** этой логики нет: воркер всегда использует Tesseract; скрипт вынесен в `scripts/local_only/` для локального запуска без влияния на VPS.
